# Mendelian Randomization Study on Adiposity and T2D

This repository contains the R code used for the Mendelian randomization analyses in our study:

"Central adiposity rather than overall obesity drives the causal risk of type 2 diabetes: a multivariable Mendelian randomization study"

## Data Sources

- BMI: ukb-b-2303
- Waist circumference: ukb-b-9405
- Hip circumference: ukb-b-15590
- Type 2 diabetes: ieu-a-24

All data are publicly available from the IEU OpenGWAS database.

## Methods

Analyses were conducted using R (version 4.x) and the TwoSampleMR package.

## Notes

This repository is provided for reproducibility purposes.
